Shopper - 실시간 소통의 온라인 쇼핑몰

2023.12 ~ 2024.06 2인(프론트엔드 1명, 백엔드 1명)

쇼핑하는 사람을 위한 온라인 쇼핑몰 프로젝트로 Shop + per (쇼핑하는 사람)으로 서비스명을 짓게 되었습니다.
Shopper는 실시간 대화 기능으로 사용자 간 의견 전달을 용이하게 하여, 소통을 강화한 온라인 쇼핑몰 서비스입니다.
온라인 쇼핑몰에 실시간 채팅 시스템을 만들어 사용자 간 정보들을 공유하는 환경 제공하면 좋겠다는 생각으로 시작되었습니다.

온라인 쇼핑몰에서는 사용자들끼리 정보를 공유하는 실시간 정보 공유 기능이 필요하다고 생각됩니다.
저희의 목표는 사용자들이 간편하게 쇼핑 아이템, 사이트 문제점, 관리자 피드백 등등에 대한 의견을 쉽게 나눌 수있는 서비스를 제공하는 것입니다.

*현재 프로젝트가 종료되어 수정이 어려운 부분이 많은 상태입니다.

📍 주요 기능 및 특징

  • 상품 등록 및 수정
  • 결제 API를 활용해서 상품 결제
  • 마이페이지에서 구매한 상품 확인/환불/취소
  • 실시간 채팅 시스템을 만들어 사용자 간 정보들을 공유하는 환경 제공
  • 상담원 연동 기능으로 고객의 의견을 빠르게 수집하고 관리

🛠️ 사용 기술 및 언어

  • 개발 툴: Visual Studio, IntelliJ
  • 프레임워크: Spring
  • 운영체제: Windows
  • 라이브러리: React, Spring Data JPA
  • 형상관리 및 배포: AWS, GitHub
  • DB: MySQL

☕️ 검토

팀원과의 스팩 차이 및 프론트엔드에 관한 지식 부족
  • 함께 진행하는 팀원과 비교했을 때 기술 스팩 면에서 부족함을 느꼈습니다. 비록, 팀원은 백엔드, 저는 프론트엔드로 기술 부분이 달랐지만, 프론트 기술 부분에서 많은 도움을 줘서 더욱 발전해야겠다고 생각하였습니다.
  • 이를 극복하기 위해 메타인지 방식을 활용해 현재 제가 부족한 점이 무엇이고, 팀 프로젝트에서 어떤 역할을 수행할 수 있는지 고민했습니다.
  • 부족한 기술 분야를 보완하기 위해 개인 학습 시간을 가졌습니다. 또한, 작업 시간을 남들 보다 더 투자하여 맡은 역할을 수행했으며, 모르는 부분은 팀원들에게 질문하고 협업을 통해 적극적으로 도움을 받았습니다.
  • 더불어 HTML, CSS, 디자인 강점을 발휘해 프로젝트 완성도를 높이고, 팀원과 함께 해결하며 서로에게 배우고 성장할 수 있었습니다.
  • 이번 프로젝트를 통해 꾸준한 자기 계발과 새로운 기술 트렌드를 학습하기 위해 스스로 찾아보고있습니다.

  • 형상관리 및 배포: GitHub

✨ 작업 기여도

포트폴리오 제작 의도를 파악하고 기획 및 구체화 작업

  • 포트폴리오를 보는 사용자들에게 원하는 정보를 쉽게 전달할 수 있도록 고려

  • green & gray를 활용해 심플하면서도 차분한 느낌을 주었으며, 익숙한 배치 등 UX 측면을 고려하며 사용자 경험을 개선하기 위해 노력 중
웹 표준 / 접근성 고려 및 인터랙션 작업
